# 機能・使用技術一覧
 - 技術一覧
    - 言語/フレームワーク
      - Ruby2.5.3
      - Ruby on Rails5.2.2
      - Bootstrap4
- データベース
  - MySQL
- 機能一覧
  - ユーザ登録、ユーザログイン機能
  - 投稿一覧表示機能、投稿詳細表示機能
  - 投稿へのコメント機能
  - 画像ファイルのアップロード機能
  - ダイレクトメッセージ(DM)機能
  - 通知機能（DMと投稿へのコメントを通知）
  - ページネーション機能
  - 単体テスト(Rspec)
